# 程序员如何高效地写技术文章

写技术文章是程序员最有价值的投资之一。

## 为什么要写博客？

**费曼学习法**的核心是：你只有能够用简单的语言解释一件事，你才真正理解了它。

## 如何选题

好的选题来源于**自己的真实需求**：

- 最近解决了一个棘手的 bug？写下来
- 学了一个新工具感觉很有用？写篇介绍
- 对某个设计决策有独到看法？分享出来

## 文章结构

```
1. 痛点 / 背景
2. 核心概念
3. 实战示例
4. 总结 / 延伸阅读
```

## 关于坚持

每周一篇是个很好的节奏。**发出去的不完美文章，比草稿里的完美文章有价值一万倍**。

> 开始写吧，你的第一篇文章不需要完美，只需要存在。
